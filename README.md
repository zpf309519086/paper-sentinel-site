# 文献哨兵官网

文献哨兵（Paper Sentinel）的公开产品网站，包含产品介绍、隐私政策、用户条款和使用支持页面。

文献哨兵是一款本地优先的 Edge 浏览器扩展。它在中国知网和 Web of Science 文献页面中，核对用户自己选择的 Excel 文献清单与本地论文文件夹，并提示文献是否已经收藏、是否已经下载以及对应的文献编号。

## 页面

- `/`：产品介绍
- `/privacy`：隐私政策
- `/terms`：用户条款
- `/support`：使用支持

## 本地运行

需要 Node.js 22.13.0 或更高版本。

```bash
npm install
npm run dev
```

生产检查：

```bash
npm test
```

## 隐私原则

网站不收集文献数据。当前扩展版本在本机处理题录、文献清单和文件夹索引，不上传论文正文、Excel 内容、文件名、文件路径或浏览记录。详细说明见网站的隐私政策页面。
